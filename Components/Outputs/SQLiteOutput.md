# SQLite Output
## Settings
- Store
	- Accessible path to write the SQLite file
	- No default, **Must** be present or component will take no action
- Commit Time
    - Updates per second
    - Default, 1
- Enable Monitoring
    - Toggles monitoring for the SQLite connection
    - Default, enabled
- Record at Start
    - Toggles recording at start
    - Default, enabled
