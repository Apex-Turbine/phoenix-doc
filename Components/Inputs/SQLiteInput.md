# SQLite Input
## Settings
- SQLite Path
	- Accesible path to the SQLite file
	- No default, **Must** be present or component will take no action
- Start Time
    - Start date and time for the query
    - Default, disabled
- End Time
    - End date and time for the query
    - Default, disabled
- Only Data Points
    - Filtering for data points only
    - Default, disabled
