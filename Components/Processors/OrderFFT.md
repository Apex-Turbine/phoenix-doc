# Order FFT
## Settings
- Block Size
	- Angle-Domain Block Size
	- Default, 1024

- Scaling
	- Spectrum Scaling Mode
	- Default, P2P

- Window
	- Function
	- Options: None, Hamming, Nuttall, Blackman, Flat-Top, Blackman-Harris
	- Default, Blakman-Harris

- Order Resolution
	- Default, 0.500

- Orders
	- List of orders to output
	- Default, All (blank)

- Max Order
    - **Read-only display**
    - Based on Block Size and Order Resolution
    - Default, 256
